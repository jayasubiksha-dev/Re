# Student Registration Form

A responsive and interactive Student Registration Form built using HTML and CSS.

## Features

- User-friendly registration form
- Input fields for:
  - Full Name
  - Email
  - Password
  - Confirm Password
  - Gender Selection
  - Date of Birth
  - Department Selection
  - Skills Selection
  - About Yourself Section
- Terms and Conditions checkbox
- Register and Reset buttons

## CSS Features

- Linear Gradient Background
- Flexbox-based Form Alignment
- Hover Effects on:
  - Input Fields
  - Select Menu
  - Textarea
  - Buttons
- Smooth Transitions
- Form Shadow Effects
- Floating Heading Animation
- Fade-In Animation
- Rounded Corners and Modern UI Design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Animations
- CSS Transitions
